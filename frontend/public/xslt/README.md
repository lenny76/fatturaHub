# Fogli di Stile Ufficiali (XSLT)

Scaricare i fogli di stile ufficiali dall'Agenzia delle Entrate e posizionarli in questa cartella.

## File richiesti

- `FatturaPA_v1.2.3.xsl` — FatturaPA (verso PA)
- `FatturaOrdinaria_v1.2.3.xsl` — Fattura Ordinaria (B2B/B2C)
- `FatturaSemplificata_v1.0.2.xsl` — Fattura semplificata

## Download

I file sono disponibili sul sito ufficiale:
**https://www.fatturapa.gov.it** → Norme e regole → Documentazione → Fogli di stile

Oppure dal sito dell'Agenzia delle Entrate nella sezione fatturazione elettronica.

> Senza il file XSLT, il visualizzatore usa automaticamente un renderer HTML semplificato integrato.
